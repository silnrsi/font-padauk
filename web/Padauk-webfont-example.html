Using Padauk webfonts

The page demonstrates how to use the WOFF2 version of Padauk for web pages, using local copies of the .woff2 files in this archive. For more information on this technique - and how to use the normal TrueType version as a web font - see Using SIL Fonts on Web Pages.

The body of this document is styled using an external stylesheet, with Padauk set as the default font, and loaded directly as a WOFF2 file using the @font-face rule. If you're seeing a sans-serif font here and the Padauk typeface in the next paragraph, then it is working. Otherwise the text will be in a serif font or your browser default. This technique should work in Firefox, Chrome, Safari and Edge.

Padauk user-selectable features

The Padauk font includes a number of optional features that provide alternative rendering that might be preferable for use in some contexts. The examples below show the results of applying these features. Whether these features are available to users will depend on both the application and the rendering technology (Graphite or OpenType) being used. Graphite features are available in Firefox. The OpenType features are used in other browsers. Further documentation of these feature is available in the documentation folder of the font download.

To see specifically how these techniques are specified in the CSS and HTML, view the source to this page and the accompanying CSS file.
