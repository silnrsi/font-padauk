Padauk - Font Features

Padauk is an OpenType-enabled font family that supports the Myanmar (Burmese) script. It includes a number of optional features that may be useful or required for particular uses or languages. This document lists all the available features.

These OpenType features are primarily specified using four-letter tags (e.g. 'ss01'). For more information on how to access OpenType features in specific environments and applications, see Using Font Features.

Please note that Graphite support has been removed in the current release, but continues to be available in the version 5.100 fonts. See our Previous Versions archive.

This page uses web fonts (WOFF2) to demonstrate font features and should display correctly in all modern browsers. For a more concise example of how to use Padauk as a web font see ../web/Padauk-webfont-example.html in the font package web folder. For detailed information see Using SIL Fonts on Web Pages.

MyanmarWeb also documents how to use the fonts on the web.

If this document is not displaying correctly a PDF version is also provided in the documentation/pdf folder of the release package.
